# Support

This repository contains the X07 package registry server (API).

## Questions / troubleshooting

If you're unsure how to publish a package or how the registry fits into the toolchain workflow,
please start with:

- X07 docs: https://x07lang.org/docs/packages/
- GitHub Discussions (recommended): https://github.com/x07lang/x07/discussions

## Bug reports

GitHub Issues in this repo are for actionable bugs in the registry API:

- Include repro steps.
- Include request_id / x-request-id headers if present.
- Include relevant JSON responses (redact tokens).

## Security issues

Do not open public issues for security problems.

See [SECURITY.md](SECURITY.md).
